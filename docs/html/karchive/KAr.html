KAr Class Reference

from PyKDE5.karchive import *

Inherits: KArchive

Detailed Description

KAr is a class for reading archives in ar format. Writing is not supported. A class for reading ar archives.

Author:
Laurence Anderson <l.d.anderson@warwick.ac.uk>
